HTML

标签

<a>中不能再嵌套<a>
<ul>中嵌套<li>构成无序列表 Unorderlist
<ol>中嵌套<li>构成有序列表 Orderlist
<dl>标签定义一个描述列表，与<dt>(定义项目名字)和<dd>(描述项目)一起使用
用法：(<dt>和<dd>可为任意个)

HTML5中新的表单元素

  • datalist : 规定输入域的选项列表,类似于下拉框

  • output : 用于不同类型的输出

  • keygen : 用于验证用户，但是已移出标准

置换元素

  • 置换元素：浏览器根据元素的标签内容，来决定元素具体显示的内容
    常见的置换元素有：<img>、<input>、<textarea>、<select>、<object>

  • 不可置换元素：内容直接表现给客户端（如浏览器）
    常见的不可置换元素有：<label>、<a> 等

CSS

link和@import的区别

(1) link属于HTML标签，而@import由CSS提供
(2) 页面加载时，link会同时被加载，而@import引用的CSS会等到页面加载完时再加载
(3) import的IE版本要求为>IE5，而link无兼容问题

字体相关的CSS

可继承的样式

  • 可继承:font-size,font-weight,line-weight,color,cursor
  • 不可继承的一般是会会改变盒子模型的:display,margin,border,padding,height等
  • 如果不想要继承属性的话可以设置position: static,这样做还会导致top,left,right失效

JS获取CSS

getComputedStyle可以获取渲染结束后最终的样式

null和undefined的异同

相同点：

  • 在if判断语句中，值默认为false
    差异：
  • null转化为数字类型为0，而undefined转化为数字类型为NaN
  • null是一个很特殊的对象，常见做法为作为参数传入
  • 设置为null的变量或者对象会被内存收集器回收
